

## План изменений

### 1. Телефонный ввод с автоматическим +7 и выбором кода страны

Создать компонент `PhoneInput` с:
- Выпадающий селектор кода страны (🇷🇺 +7, 🇧🇾 +375, 🇰🇿 +7, 🇺🇦 +380 и т.д.), по умолчанию +7
- Автоматическая маска ввода номера
- Использовать в `Cart.tsx` (форма заявки) вместо обычного `<input type="tel">`

### 2. Прокрутка наверх при переходе между страницами

**Проблема**: React Router не прокручивает страницу наверх при навигации — пользователь видит низ страницы.

**Решение**: Добавить компонент `ScrollToTop` в `Layout.tsx`, который вызывает `window.scrollTo(0, 0)` при каждой смене маршрута (`useLocation`). Это исправит поведение для каталога, корзины и всех остальных страниц.
