Для создания пользователя в системе аутентификации нужна edge-функция с правами сервисного ключа.

### Шаги

1. **Создать edge-функцию `create-admin`** — она использует Supabase Admin API для:
   - Создания пользователя с email `k.aya@kaya.com` и временным паролем
   - Добавления записи в таблицу `user_roles` с ролью `admin`

2. **Вызвать функцию** для создания пользователя

3. **Удалить edge-функцию** после использования (одноразовая)
