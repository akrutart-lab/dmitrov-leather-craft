

## Изменения

### 1. PhoneInput в чат-боте
Заменить `<Input placeholder="Телефон" type="tel" ...>` (строка 299) на `<PhoneInput>` компонент с автоматическим +7 и выбором страны. Инициализировать `phone` как `'+7'` вместо пустой строки.

### 2. Сообщения открываются сначала
Сейчас чат скроллит вниз (`scrollToBottom`). Нужно при открытии чата (`setOpen(true)`) скроллить контейнер сообщений наверх, чтобы первое сообщение было видно. Добавить `useEffect` на `open` — когда чат открывается, скроллить контейнер наверх. `scrollToBottom` оставить только для новых сообщений (когда пользователь пишет).
